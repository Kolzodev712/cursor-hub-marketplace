## [0.1.0] - Initial release

### Added

- **Marketplace**
  - Multi-plugin repo with manifest at `.cursor-plugin/marketplace.json`.
  - Validation script `npm run validate` and JSON schemas in `schemas/`.

- **design-log-foundation**
  - Design-log methodology and manual create/record-step commands.
  - Foundation for workflow recording; pairs with language workflows.

- **rust-workflow**
  - Full Rust workflow: design review, implementation, testing, bugfix, PR review.
  - Documentation and security commands.

- **python-workflow**
  - Full Python workflow: design review, implementation, testing, bugfix, PR review.
  - Documentation and security commands.

- **js-ts-workflow**
  - Full JS/TS workflow: design review, implementation, testing, bugfix, PR review.
  - Documentation and security commands.

- **terraform-workflow**
  - Full Terraform workflow: design review, implementation, testing, bugfix, PR review.
  - Documentation and security commands.
